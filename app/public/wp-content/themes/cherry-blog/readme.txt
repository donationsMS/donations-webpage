== Description ==
Cherry Blog is modern style free responsive WordPress theme. Cherry Blog can be used for photography, fashion, news, music, entertainment, personal blog sites and almost any kind of sites. It is SEO friendly with featured image supports and custom colors. Theme customizer can be used to change color, upload logo and other theme settings. You can get quick SUPPORT through support page https://99colorthemes.com/support/ and view live DEMO at https://demo.99colorthemes.com/cherry-blog

== Installation ==
1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
